## Korean
만능 서치 헬프

### 설명
자동으로 테이블을 찾아서 서치 헬프를 띄워 줍니다.

우선 데이터 엘리먼트에 연결된 도메인에 연결된 밸류 테이블 에서 검색합니다.

밸류 테이블이 없으면 화면에 필드 선언할때 사용한 테이블-필드의 테이블을 찾아서 검색합니다.

### 필요 사항
ABAP 버전: 750 이상

데이터베이스: HDB(HANA)
